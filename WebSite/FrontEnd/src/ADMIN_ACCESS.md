# BridgeUS Admin Dashboard Access

## How to Access Admin Panel

To access the admin dashboard, follow these steps:

1. Navigate to your **Profile** page (click "Profile" in the navigation bar)
2. Scroll down to find the **"Admin Access"** button in the sidebar (Desktop view)
3. Click the button to enter Admin Mode

On mobile, the Admin Access button is currently only available in desktop view. For full admin functionality, please use a desktop or tablet device.

## Security Note

In a production environment, this admin panel would be:
- Protected by authentication and authorization
- Only accessible to users with admin role
- Secured with proper session management
- Logged for audit purposes

For this demo, the admin panel is accessible through the Profile page for demonstration purposes.

## Exit Admin Mode

To exit the admin panel and return to the regular user interface:
- Click the **"Exit Admin"** button in the top-right corner of the admin panel
