# Step 07 搜索与发现

## 目标
实现全文检索、过滤与热门内容推荐。

## 前置条件
- Step 06 完成

## 详细任务拆解
1. 搜索服务
   - PostgreSQL 全文检索或独立搜索服务
   - 支持标题/内容/标签检索
2. 筛选与排序
   - 分类、标签、语言
   - 排序：最新/最有用/最高准确
3. 搜索建议
   - 关键词补全与热门词
4. 热门与趋势
   - 互动量 + 时间衰减计算
   - 可配置权重

## 产出物
- `/search`、`/search/suggestions`、`/search/trending`

## 验收标准
- 查询结果准确
- 热门排序符合预期
- 搜索建议可用

## 风险与注意
- 多语言检索需明确分词策略
- 热门算法需防刷
