# Step 06 回复与互动评分

## 目标
完成回复体系与互动评分机制（有用度/准确度）。

## 前置条件
- Step 05 完成

## 详细任务拆解
1. 回复接口
   - 新增、更新、删除
   - 列表分页
   - 回复状态（visible/hidden）
2. 有用度投票
   - 帖子/回复投票
   - 防重复投票（唯一约束）
   - 取消投票
3. 准确度反馈
   - 使用后评分与备注
   - 一人一评
4. 统计聚合
   - 帖子/用户的 helpfulness、accuracy 汇总
   - 定时任务或触发更新

## 产出物
- `/replies/*`、`/helpful`、`/accuracy` 接口
- 互动统计逻辑

## 验收标准
- 投票后统计值正确更新
- 同用户无法重复投票
- 取消投票后统计同步更新

## 风险与注意
- 需保证投票幂等性
- 统计更新需避免并发冲突
