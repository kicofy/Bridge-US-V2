# Step 04 用户资料与认证审核

## 目标
完成用户资料管理与身份认证审核流程。

## 前置条件
- Step 03 完成

## 详细任务拆解
1. 用户资料
   - 个人资料查询与更新
   - 公共字段与私密字段分离
   - 头像上传与更新
2. 认证材料
   - 认证材料上传（files 表）
   - 认证申请提交流程
   - 申请状态流转（pending/approved/rejected）
3. 后台审核
   - 认证队列查询
   - 通过 / 拒绝 / 要求补充
   - 更新可信度与徽章
4. 权限控制
   - 认证资料仅本人/管理员可访问
   - 认证结果可公开显示

## 产出物
- `/profiles/*` 与 `/verification/*` 接口
- 认证审核流程打通

## 验收标准
- 用户可提交认证申请
- 管理员审核后用户可信度更新
- 非管理员无法访问他人认证资料

## 风险与注意
- 认证资料涉及 PII，需隔离存储
- 审核操作必须记录审计日志
