# Step 08 审核举报与管理后台

## 目标
完成举报审核流程与后台管理功能。

## 前置条件
- Step 07 完成

## 详细任务拆解
1. 举报流程
   - 举报创建、查询、处理
   - 举报原因与证据记录
2. 审核队列
   - 内容待审列表
   - 审核操作日志
   - 审核状态流转
3. 管理后台接口
   - 用户管理（封禁/解封）
   - 帖子/回复管理（下架/恢复）
   - 认证申请管理
   - 分类/标签管理
4. 权限与审计
   - 仅管理员/版主可访问
   - 操作日志可追溯

## 产出物
- `/reports/*`、`/moderation/*`、`/admin/*` 接口
- 审核日志记录机制

## 验收标准
- 举报处理影响内容状态
- 管理员可执行所有管理操作
- 审核行为记录完整

## 风险与注意
- 审核权限控制要严格
- 审核操作需支持回滚策略
